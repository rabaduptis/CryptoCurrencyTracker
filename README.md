### CryptoCurrency Tracker App

- Support lists all cryptocurrencies
- All the details about the selected cryptocurrency
- Firebase Auth
- Define the refresh interval of current price
- Favorite coin list
- Loading screen for large coin lists

#### Features
 ------------
- MVVM design pattern
- Retrofit, Glider, Hilt, Room, Coroutines
- Various Android jetpack libraries (such as NavigationController)

#### Features that are discontinued or need to be developed
- Background worker, dependency injection issues
- Favorite coin system should be improved
